Detailed Description

EoaBufferU16 is a buffer of 16 bit long characters where each character is represented by UInt16. EoaBufferU16 is used to contain UTF16 data.

Remarks:
EoaBufferU16 class contains a copy of the buffer passed on set methods.
All methods in this class are Lock-free.
