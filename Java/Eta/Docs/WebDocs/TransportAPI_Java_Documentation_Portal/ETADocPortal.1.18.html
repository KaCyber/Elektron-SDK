Transport API Java Edition

Value Added Components Product Features
The Transport API Valued Added Components contain the following components:
Transport API Reactor Component
See also Transport API Reactor Component Reference Manual
•
This is a callback-based component that provides connection management and event processing features for OMM-based applications.
•
Supports OMM-based Consumers, Providers and Non-Interactive Providers.
•
Provides domain-specific callbacks for the Login, Source Directory and Dictionary RDM representations.
Administrative Domain Model Representations
See also Administrative Domain Model Representations Reference Manual
•
This component provides RDM-specific representations of the OMM Administrative domain models.
•
Provides classes that represent specific messages defined in the Transport API Java Edition RDM Usage Guide for the Login, Source Directory and Dictionary domains.
•
In addition, this component handles encoding and decoding of the OMM Administrative domain messages, which significantly reduces the amount of code necessary to interact with those domain models.
Value Added Utilities
See also the Value Added Reference Manual .
This component is a collection of common classes, mainly used by the Transport API Reactor.
