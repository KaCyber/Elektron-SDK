What is the Transport API Java Edition?
The Transport API Java Edition is the Java programming language release of Thomson Reutersâ€™s low-level API. It allows clients to write applications for use on the Thomson Reuters Enterprise Platform (TREP) with the highest performance, highest throughput, and lowest latency.
The Transport API is a custom API that supports the OMM message model. It complements other APIs by allowing users to choose the type of functionality and layer at which they want to access the TREP. With the addition of the Transport API, customers now have a choice between a feature loaded session level API (RFA) and a high performance transport level API (the Transport API).
