Transport API Value Added Components Functionality
This release also includes the Transport API Value Added components. These components are intended to simplify Transport API usage, and reduce the amount of code required to create OMM-based applications. In addition to the previous Transport API features, these components provide the following additional features:
•
The Transport API Reactor component is a connection management and event processing component.
•
The Administration Domain Model Representations reduce the code required to interact with the RDM administrative domains.
•
The Value Added Utilities are a collection of common classes mainly used by the Transport API Reactor.
