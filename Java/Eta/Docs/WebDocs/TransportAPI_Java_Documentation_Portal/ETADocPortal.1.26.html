Value Added Component Examples
Consumer
This is an example of an OMM Consumer type application written with the Transport API ValueAdd library and demonstrates how to consume data from an OMM Provider using Value Add Components.
NIProvider
This is an example of an OMM Non-Interactive Provider application written with the Transport API ValueAdd Library and demonstrates how to non-interactively provide Level I Market Price data to an Advanced Data Hub(ADH) using Value Add Components.
VAProvider
This is an example of an OMM Interactive Provider application written with the Transport API ValueAdd Library and demonstrates how to interactively provide data to OMM Consumer applications using Value Add components.
