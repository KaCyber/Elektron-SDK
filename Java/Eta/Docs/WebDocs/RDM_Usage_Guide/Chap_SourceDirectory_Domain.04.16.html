Source Directory ConsumerStatus Generic Message Payload
 
Note: GenericMsg(s) are supported for the DIRECTORY RDM only for sending / receiving information related to ConsumerStatus/Source Mirroring Mode.
The data structure for the ConsumerStatus message is a Map. Each MapEntry sends status to one service and is uniquely identified by serviceId (its key). Each entry contains an ElementList with one ElementEntry indicating how the provider is used.
