Field Dictionary
Field Dictionary Payload
The payload of a Field Dictionary Refresh Message consists of an Series where each series entries contains an ElementList. Each SeriesEntry represents a row of information in the dictionary. The ElementList contained in each series entry provides information about an element of the row.
