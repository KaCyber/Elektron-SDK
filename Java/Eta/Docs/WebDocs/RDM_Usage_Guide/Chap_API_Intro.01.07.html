Conventions
•
The Reuters Domain Models (RDMs) are described in terms of OMM concepts. Images and XML example layouts are provided as a reference in relavant sections.
•
Inline Classes, methods, and types are shown in orange, Lucida Console font.
•
Parameters, filenames, and directories are shown in Bold font.
•
Document titles and variable values are shown in italics.
•
When included in the body of the text, new concepts are called out in Bold, Italics the first time they are mentioned.
•
Verbose code examples (one or more lines of code) are show in Lucida Console font against an orange background. Comments in such code samples are formatted in green coloring. For example:
