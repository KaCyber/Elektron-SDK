Conventions
Typographic Conventions
•
Directories, filenames, parameters, and list entries appear in bold; e.g.: filename
•
When in text, keywords, function names, and method names appear in orange, Courier New font.
•
Method names omit their arguments unless the arguments are needed for clarity.
•
Variables and document titles appear in italics, e.g.: Transport API C Edition Developers Guide
•
Code samples appear in a Courier New font with an orange background; e.g.:
 
 
typedef struct
{
    unsigned char flags;
    unsigned char code;
} SSL_STATUS_CODE;
