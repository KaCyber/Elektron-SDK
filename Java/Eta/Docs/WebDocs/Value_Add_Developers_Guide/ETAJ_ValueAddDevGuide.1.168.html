DESCRIPTION

Optional. Indicates the application ID.
If present, a flags value of LoginAttribFlags.HAS_APPLICATION_ID should be specified.

If populated in a login refresh, applicationId should match the applicationId used in the login request.

Optional. Indicates the application name.
If present, a flags value of LoginAttribFlags.HAS_APPLICATION_NAME should be specified.

position
Optional. Indicates the DACS position.
