Dictionary Encoding and Decoding
FUNCTION NAME
DESCRIPTION
encode
Encodes a dictionary message. This method takes the EncodeIterator as a parameter into which the encoded content is populated.
decode
Decodes a dictionary message. The decoded message may refer to encoded data from the original message. If the message is to be stored for later use, use the copy method of the decoded message to create a full copy..
