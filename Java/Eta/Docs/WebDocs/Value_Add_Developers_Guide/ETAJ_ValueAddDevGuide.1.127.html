Tunnel Stream Submit
The submit method writes content to the TunnelStream. This method also enforces any specified behaviours on the content being submitted (e.g., if guaranteed messaging is specified, this content will follow all configured persistence options).
METHOD NAME
DESCRIPTION
submit
Use the submit method to pass in opaque or RDM Message content (including Queue Messages) to be processed and sent over the TunnelStream.
This method has additional options that can be specified via TunnelStreamSubmitOptions (refer to Table 74).
