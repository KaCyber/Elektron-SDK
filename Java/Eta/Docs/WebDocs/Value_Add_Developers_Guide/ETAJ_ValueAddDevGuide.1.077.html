Removing Reactor Channels
ReactorChannel.close Method
The following method can be used to remove a ReactorChannel from a Reactor instance. It will also close and clean up any resources associated with the ReactorChannel.
METHOD NAME
DESCRIPTION
ReactorChannel.close
Removes a ReactorChannel from the corresponding Reactor and cleans up associated resources. This will additionally invoke the Channel.close method, as described in the Transport API Java Developers Guide, to clean up any resources associated with the underlying Channel.
This method can be called from either outside or within a callback.
