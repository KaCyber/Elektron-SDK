Directory Refresh Flag Enumeration Values
FLAG ENUMERATION
MEANING
HAS_SERVICE_ID
Indicates the presence of serviceId.
SOLICITED
If this flag is present, it indicates that the login refresh is solicited (e.g., it is in response to a request). If the flag is not present, this refresh is unsolicited.
HAS_SEQ_NUM
Indicates the presence of sequenceNumber.
CLEAR_CACHE
Indicates that stored payload information associated with the login stream should be cleared. This might happen if data is known to be invalid.
