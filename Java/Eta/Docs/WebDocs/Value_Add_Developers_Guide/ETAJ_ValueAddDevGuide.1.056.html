Reactor Creation
The creation of a Reactor instance can be accomplished through the use of the following method.
METHOD NAME
DESCRIPTION
ReactorFactory.createReactor
Creates a Reactor instance, including all necessary internal memory and threads. Once the Reactor is created, ReactorChannels can be associated, as described in Section 6.3.
Options are passed in via the ReactorOptions, as defined in Table 5.
