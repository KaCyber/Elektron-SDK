ShmemOpts Method
 
Table 18: ShmemOpts Method Option  
METHOD
DESCRIPTION
maxReaderLag
Maximum number of messages that the client can have waiting to be read. If the client "lags" the server by more than this amount, the client will be disconnected on its next attempt to read. The default is equal to 75% of the number of buffers in the shared memory segment.
