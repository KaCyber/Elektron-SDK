PostUserInfo Methods
 
Table 140: PostUserRights Methods  
Method
DESCRIPTION
userId
Sets or gets the userId, which identifies the specific user that posted this data.
userAddr
Sets or gets the IP Address (userAddr) of the user that posted this data. Though the address can be specified as either a long or String (e.g. "127.0.0.1"), if it is specified as a String, it will be converted to its integer equivalent.
userAddrToString
Converts an IP address in integer format to its string equivalent.
clear
Clears the object, so that it can be reused.
