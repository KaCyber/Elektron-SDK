Series
The Series is a uniform container type. Each entry, known as an SeriesEntry, contains only encoded data. This container is often used to represent table-based information, where no explicit indexing is present or required. A Series can contain zero to N1 entries, where zero entries indicates an empty Series.
1
A Series currently has a maximum entry count of 65,535. This type has an approximate maximum encoded length of 4 gigabytes but may be limited to 65,535 bytes if housed inside of a container entry. The content of an SeriesEntry has a maximum encoded length of 65,535 bytes. These limitations can change in subsequent releases.
